# react-template

> 项目模板需要在[安全上下文](https://developer.mozilla.org/en-US/docs/Web/Security/Secure_Contexts)中运行。

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload
npm run serve

# build for production with minification
npm run build

# run all tests
npm test
```
