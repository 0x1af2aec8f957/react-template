# react-template

> A React.js template [Already eject].

> Use [bulma.css](https://bulma.io) (Old version use qc_ui).

> Build use [parceljs](https://parceljs.org).

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload
npm run serve

# build for production with minification
npm run build

# run all tests
npm test
```
